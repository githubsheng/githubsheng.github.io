Powerful search features.

  • Powerful auto-complete feature.
  • You can search by multiple key words
  • Ranks the results based on various kinds of factors. Make sure the most relevant ones always stay on top of the result list
  • You can right click on any search result to view the note on the right half of the screen, and still keep the search result on the left half of the screen.

Use markdown to effortlessly generate beautiful styles

Available markdowns are:
  • *: bold
  • _: italic
  • `: inline code block
  • ~: text in blue color
  • @header: text as header
  • @line: draw a horizontal line
  • @important: text in blue background
  • @less: text in lightgray background
  • @js: create a JS code block. Code inside is syntax highlighted
  • #: create a tag
  • `note-ref:`: generate a hyper link to another note

Easily generate tags to organize your notes

You can later search notes by their tags. Tags have strong influence on the ranking of the search results. If a note contains one or more tags, and the tags are search keywords at the same time, then this note will typically rank very high in the search result.

Easily generate a hyper link to another note

  • Every note has a reference Id displayed alongside its title. You can use markdown `note-ref: reference id` to create a hyper link to a note
  • You can easily navigate backwards by clicking on the "back" button on the left bottom corner.

Drag and drop one or more images to your note and resize them

All the inserted images will be immediately persisted

Draw on the inserted images!

  • You can draw on top of the inserted images
  • You can choose your line width and color
  • Made a mistake? You can use eraser to undo the drawing

System will automatically find out related notes for you

  • Lists all the notes that current references
  • Lists all the notes referenced by current note
  • Lists all notes whose content are similar to the current note
  • You can click on any of the links to navigate to another note, and get back by clicking on the "back button" on the left bottom corner
